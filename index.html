About me
I am a software engineer interested in high-impact projects related to systems security, application porting and low-level internals of programming language runtimes. I have extended experience in TEEs (e.g. Intel SGX, Arm TrustZone) and the runtime environments of high-level programming languages such as JavaScript and Lua.
Notable Work
Dynamic code execution on Intel SGX nodes deployed on the cloud. Several improvements are implemented including faster bootstrap, I/O and battery consumption.
  • Porting of QuickJS interpreter within Intel SGX enclave
  • Port to embedded devices and different architectures (Raspberry, Android, x86)
  • Wrappers and proxies of the enclave code with the untrusted OS
  • Enhanced QuickJS with cryptographic and network capabilities
  • Client/Server code to establish end-to-end communication and offload requests
PaSh is a system for parallelizing POSIX shell scripts. It has been shown to achieve order-of-magnitude performance improvements on shell scripts.
  • Automation scripts for correctness and performance tests based on Github Actions
  • Rendering and evaluation infrastructure across Github commits
  • Automation scripts for building Docker images for different Linux flavors
System that simplifies the development of confidential computing. It solves several TEE challenges such as the lack of high-level TEE abstractions, runtime extensibility and Lua application porting.
  • Porting of Lua5.4 interpreter within Intel SGX enclave
  • Wrappers and proxies of the enclave code with the untrusted OS
  • Client/Server code to establish end-to-end communication and offload requests
  • Porting of Lua interpreter within Arm TrustZone
First port of Intel SGX to Android x86 Ecosystem.
  • Ported Intel SGX framework, libraries, modules dependencies to AOSP x86 port
  • Developed and enhanced popular Android services (Keystore, Pairing protocols) with enclave capabilities
  • Offer a Java API for facilitating the use of SGX features in Android applications
  • Offer a toolchain, fully compliant with Android and Intel SGX
