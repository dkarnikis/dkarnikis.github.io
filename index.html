About me
I am a software engineer interested in high-impact research projects related to systems security, application porting and low-level internals of programming language runtimes. I have extended experience in TEEs (e.g. Intel SGX, Arm TrustZone) and the runtime environments of high-level programming languages such as JavaScript and Lua.
Notable Work
Distributed Code Execution on Javascript atop of TEEs
Dynamic code execution on Intel SGX nodes deployed on the cloud. Several improvements are implemented including faster bootstrap, I/O and battery consumption.
  • Porting of QuickJS interpreter within the Intel SGX enclave
  • Porting of the tool to embedded devices and different architectures (Raspberry, Android, x86)
  • Wrappers and proxies of the enclave code with the untrusted OS
  • Enhanced QuickJS with cryptographic and network capabilities
  • Client/Server code to setup the communication channel and offload requests using scheduling algorithms
PaSh: Light-touch Data-Parallel Shell Processing (Open Source)
PaSh is a system for parallelizing POSIX shell scripts. It has been shown to achieve order-of-magnitude performance improvements on shell scripts.
  • Automation scripts for correctness and performance tests based on Github Actions
  • Rendering and evaluation infrastructure for every Github commit
LuaGuardia: A Confidential Computing Framework for TEEs
System that simplifies the development of confidential computing. It solves several TEE challenges such as the lack of high-level TEE abstractions, runtime extensibility and Lua application porting.
  • Porting of Lua interpreter within the Intel SGX enclave
  • Wrappers and proxies of the enclave code with the untrusted OS
  • Client/Server code to setup the communication channel and offload execution requests
  • Porting of Lua interpreter within Arm TrustZone
Andromeda: Enabling Secure Enclaves For The Android Ecosystem
First port of Intel SGX to Android x86 Ecosystem.
  • Ported Intel SGX framework, libraries and dependencies to AOSP x86 port
  • Developed and enhanced existing Android services with enclave capabilities (Keystore, Pairing protocols)
